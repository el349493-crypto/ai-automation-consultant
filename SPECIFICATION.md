# AI Automation Consultant

## Objectif

Créer un assistant IA capable de concevoir des automatisations professionnelles pour les entreprises africaines.

## Fonctionnalités Version 1

- Comprendre le besoin du client
- Poser des questions si des informations manquent
- Générer un workflow Make
- Générer un workflow n8n
- Générer les prompts IA
- Générer la documentation d'installation
- Expliquer chaque étape
- Donner les avantages et les limites

- Banques
