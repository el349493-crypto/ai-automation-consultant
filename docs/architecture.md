## Modules

### 1. Agent Consultant
Analyse les processus métier et produit un rapport.

### 2. Workflow Generator
Génère automatiquement les workflows Make et n8n.

### 3. Prompt Library
Contient des centaines de modèles de prompts réutilisables.

### 4. Knowledge Base
Stocke les procédures, cas d'usage et bonnes pratiques.

### 5. Dashboard
Permet de gérer les clients, projets et automatisations.

## Vision

Devenir la référence africaine pour la conception et le déploiement d'automatisations intelligentes avec l'IA.
